Installing qunits

The easiest way to install qunits is to run:

pip install qunits

We recommend creating a virtual environment before installing via pip, see the official guide.

Alternatively, qunits can be installed from source by cloning or downloading from the GitHub repository or by using pip to install directly from source with:

pip install git+https://github.com/patmlr/qunits.git


Dependencies

The qunits package has the following dependencies of standard scientific packages which are automatically installed along with it when using pip:

  • NumPy
  • SciPy
