Welcome to qunits' homepage

Performant
qunits is up to 8x faster than pint and utilizes numpy to ensure fast calculations of large data samples.
Dimensional analysis
qunits enforces dimensional analysis, ensuring that operations with physical quantities are consistent.
Verbosity
qunits provides common units with prefixes statically, enabling code inspection and autocompletion.
Customizable
qunits provides a flexible framework that allows users to define their own units and operations.
