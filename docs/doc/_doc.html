qunits’s module structure

Currently, there are seven modules in qunits, divided by their purpose.

  • dimension   Angular momentum algebra and transition matrix elements.
  • unit   Angular momentum algebra and transition matrix elements.

qunits’s namespaces

For the ease of access of most of the functions implemented in qunits, the seven modules are summarized in only three namespaces,

  • qunits
  • qunits.models
  • qunits.simulate

which allows most of the functions to be accessed by simply importing qunits.
