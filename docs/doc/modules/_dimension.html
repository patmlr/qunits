qunits.dimension

The qunits.dimension module contains the dimension definitions for the unit system. The seven base dimensions are defined as orthonormal vectors in a seven-dimensional vector space.

  • Time
  • Length
  • Mass
  • ElectricCurrent
  • Temperature
  • AmountOfSubstance
  • LuminousIntensity

A special dimension is the Dimensionless dimension, which is defined as the zero vector in the seven-dimensional vector space.

  • Dimensionless

The remaining dimensions are defined as linear combinations of the base dimensions.

  • Acceleration
  • Action
  • Area
  • Capacitance
  • Charge
  • ElectricCurrent
  • ElectricField
  • Energy
  • Force
  • Frequency
  • Jerk
  • LuminousIntensity
  • MagneticInduction
  • Power
  • Pressure
  • Resistance
  • Temperature
  • Time
  • Velocity
  • Voltage
  • Volume
